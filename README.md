# Terraform-Azure-MysqlDb-WebServer
This terraform file creates azure infrastructure for a database and a webserver with the following compenents: 
1- resource group
2- virtual network 
3- public subnet 
4- private subnet
5- nat gateway 
6- network security groups with inbound and outbound ports rules for the application
7- network interface for the webserver 
8- virtual machine inside the public subnet 
9- database private zone for the mysql db
10- Mysql server inside the private subnet
